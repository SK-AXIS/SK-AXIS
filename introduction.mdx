# Introduction

SK-AXIS is an AI-powered interview system that provides automated interview evaluation and reporting.

## Features

- Automated interview process
- Speech-to-Text transcription
- Answer rewriting and evaluation
- Nonverbal behavior analysis
- Final report generation

## System Architecture

The system consists of two main components:

1. **FastAPI Backend**
   - Handles interview flow
   - Processes audio files
   - Manages evaluation pipeline

2. **Spring Backend**
   - Manages interview data
   - Handles user authentication
   - Stores evaluation results
